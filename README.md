# Weather Diary (Дневник погоды)

**Автор:** [Гаврилов Иван]

**Дата:** [2026-04-30]

## Описание

Weather Diary — это GUI‑приложение на Python для ведения дневника наблюдений за погодой. Позволяет добавлять, просматривать и фильтровать записи о погоде, сохранять данные в формате JSON.

## Возможности

* Добавление записей с указанием:
  * даты (в формате YYYY‑MM‑DD);
  * температуры (°C);
  * описания погоды;
  * наличия осадков (да/нет).
* Просмотр записей в виде таблицы.
* Фильтрация записей:
  * по дате;
  * по температуре (выше заданного значения).
* Сохранение данных в файл `weather_data.json`.
* Загрузка данных из файла при запуске.
* Валидация ввода данных.

## Требования

* Python 3.6 или выше;
* библиотека `tkinter` (входит в стандартную поставку Python);
* библиотека `json` (входит в стандартную поставку Python).

## Примеры использования
### Пример 1: Добавление записи

- Введите дату в формате YYYY-MM-DD (например, 2024-01-15).

- Укажите температуру (например, -5).

- Напишите описание (например, «Снежно, ветрено»).

- Отметьте чекбокс «Осадки», если были осадки.

- Нажмите кнопку «Добавить запись».

### Пример 2: Фильтрация

- В поле «Фильтр по дате» введите нужную дату (например, 2024-01-15).

- В поле «Фильтр по температуре (>)» введите минимальное значение (например, 0).

- Нажмите «Применить фильтры» — в таблице отобразятся записи, соответствующие условиям.

### Пример 3: Сохранение и загрузка

- Нажмите «Сохранить в JSON» — данные сохранятся в файл weather_data.json.

- Нажмите «Загрузить из JSON» — данные загрузятся из файла (если он существует).
